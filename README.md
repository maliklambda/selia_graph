# Selia Graph

A modern database that manages a directed property graph. 

![Logo](https://github.com/maliklambda/selia_graph/blob/master/assets/static/selia_graph_logo.png?raw=true)


## About


### Query-flow
Queries are parsed using the [Sypher](https://github.com/maliklambda/Sypher) query language parser. 
They are then handled by the [scy-fi](https://github.com/maliklambda/selia_graph/tree/master/scy-fi) runtime, which calls the selia-graph storage engine api. 

## Installation
The database is intended to run on a x86-Linux-machine.
